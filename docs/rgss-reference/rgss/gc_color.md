# Color


RGBA カラーのクラス。各要素は浮動小数点数 ([Float](sc_float.md)) で管理されます。

## スーパークラス


- [Object](sc_object.md)


## クラスメソッド



### Color.new(*red*, *green*, *blue*[, *alpha*])


### Color.new (RGSS3)



Color オブジェクトを生成します。*alpha* を省略した場合は 255 になります。

引数を指定しない場合の初期値は (0, 0, 0, 0) となります。 (RGSS3)

## メソッド



### set(*red*, *green*, *blue*[, *alpha*])


### set(*color*) (RGSS3)



全要素をまとめて設定します。

2 番目の形式では、別の Color オブジェクトから全要素をコピーします。 (RGSS3)

## プロパティ



### red



赤の値 (0 ～ 255) です。範囲外の値は自動で修正されます。

### green



緑の値 (0 ～ 255) です。範囲外の値は自動で修正されます。

### blue



青の値 (0 ～ 255) です。範囲外の値は自動で修正されます。

### alpha



アルファの値 (0 ～ 255) です。範囲外の値は自動で修正されます。
