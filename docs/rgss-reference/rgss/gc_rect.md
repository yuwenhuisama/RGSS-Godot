# Rect


四角形のクラス。

## スーパークラス


- [Object](sc_object.md)


## クラスメソッド



### Rect.new(*x*, *y*, *width*, *height*)


### Rect.new (RGSS3)



Rect オブジェクトを生成します。

引数を指定しない場合の初期値は (0, 0, 0, 0) となります。 (RGSS3)

## メソッド



### set(*x*, *y*, *width*, *height*)


### set(*rect*) (RGSS3)



全要素をまとめて設定します。

2 番目の形式では、別の Rect オブジェクトから全要素をコピーします。 (RGSS3)

### empty



全要素を 0 に設定します。

## プロパティ



### x



四角形の左上隅の X 座標です。

### y



四角形の左上隅の Y 座標です。

### width



四角形の幅です。

### height



四角形の高さです。
