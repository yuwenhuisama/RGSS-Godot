# Tone


色調のクラス。各要素は浮動小数点数 ([Float](sc_float.md)) で管理されます。

## スーパークラス


- [Object](sc_object.md)


## クラスメソッド



### Tone.new(*red*, *green*, *blue*[, *gray*])


### Tone.new (RGSS3)



Tone オブジェクトを生成します。*gray* を省略した場合は 0 になります。

引数を指定しない場合の初期値は (0, 0, 0, 0) となります。 (RGSS3)

## メソッド



### set(*red*, *green*, *blue*[, *gray*])


### set(*tone*) (RGSS3)



全要素をまとめて設定します。

2 番目の形式では、別の Tone オブジェクトから全要素をコピーします。 (RGSS3)

## プロパティ



### red



赤成分のカラーバランス調整値 (-255 ～ 255) です。 範囲外の値は自動で修正されます。

### green



緑成分のカラーバランス調整値 (-255 ～ 255) です。 範囲外の値は自動で修正されます。

### blue



青成分のカラーバランス調整値 (-255 ～ 255) です。 範囲外の値は自動で修正されます。

### gray



グレースケール化フィルタの強さ (0 ～ 255) です。 範囲外の値は自動で修正されます。

この値が 0 以外の場合、色成分のバランス調整だけの場合よりも 余分に処理時間がかかります。
