# Window


ゲーム内のウィンドウのクラス。内部的には複数のスプライトで構成されています。

## スーパークラス


- [Object](sc_object.md)


## クラスメソッド



### Window.new([*x*, *y*, *width*, *height*]) (RGSS3)



Window オブジェクトを生成します。必要に応じて位置とサイズを指定します。

## メソッド



### dispose



ウィンドウを解放します。すでに解放されている場合は何も行いません。

### disposed?



ウィンドウがすでに解放されている場合に真を返します。

### update



カーソルの点滅、ポーズサインのアニメーションを進めます。 このメソッドは、原則として 1 フレームに 1 回呼び出します。

### move(*x*, *y*, *width*, *height*) (RGSS3)



X 座標、Y 座標、幅、高さをまとめて設定します。

### open? (RGSS3)



ウィンドウが完全に開いた状態 (openness == 255) のときに真を返します。

### close? (RGSS3)



ウィンドウが完全に閉じた状態 (openness == 0) のときに真を返します。

## プロパティ



### windowskin



ウィンドウスキンとして使用するビットマップ ([Bitmap](gc_bitmap.md)) への参照です。

スキンの仕様は前作 VX とほぼ同一です。[素材規格：ウィンドウスキン詳細](../rpgvx/resource.md#windowskin)を参照してください。

### contents



ウィンドウ内容として表示するビットマップ ([Bitmap](gc_bitmap.md)) への参照です。

### cursor_rect



カーソルの矩形 ([Rect](gc_rect.md)) です。

ウィンドウ内容を基準とした座標で指定します。 (RGSS3)

### viewport



ウィンドウが関連付けられているビューポート ([Viewport](gc_viewport.md)) への参照です。

### active



カーソルの点滅状態です。真のとき点滅します。初期値は true です。

### visible



ウィンドウの可視状態です。真のとき可視になります。初期値は true です。

### arrows_visible (RGSS3)



スクロール用矢印の可視状態です。真のとき可視になります。初期値は true です。

### pause



ポーズサインの可視状態です。ポーズサインとは、メッセージウィンドウの ボタン入力待ち状態を表す記号のことです。真のとき可視になります。初期値は false です。

### x



ウィンドウの X 座標です。

### y



ウィンドウの Y 座標です。

### width



ウィンドウの幅です。

### height



ウィンドウの高さです。

### z



ウィンドウの Z 座標です。この値が大きいものほど手前に表示されます。

Z 座標が同一の場合は後に生成されたオブジェクトほど手前に表示されます。

初期値は 100 (RGSS3) です。

### ox



ウィンドウ内容の転送元原点の X 座標です。この値を変化させることによって スクロールを行います。

カーソルに対しても影響します。 (RGSS3)

### oy



ウィンドウ内容の転送元原点の Y 座標です。この値を変化させることによって スクロールを行います。

カーソルに対しても影響します。 (RGSS3)

### padding (RGSS3)



ウィンドウの枠と内容の間にある余白の大きさです。初期値は 12 (RGSS3) です。

### padding_bottom (RGSS3)



下端専用の padding です。padding と同時に変更されるため、padding よりも後に設定する必要があります。

### opacity



ウィンドウの不透明度 (0 ～ 255) です。範囲外の値は自動で修正されます。初期値は 255 です。

### back_opacity



ウィンドウ背景の不透明度 (0 ～ 255) です。範囲外の値は自動で修正されます。初期値は 192 (RGSS3) です。

### contents_opacity



ウィンドウ内容の不透明度 (0 ～ 255) です。範囲外の値は自動で修正されます。初期値は 255 です。

### openness



ウィンドウのオープン度 (0 ～ 255) です。範囲外の値は自動で修正されます。

この値を 0 (完全に閉じた状態) から 255 (完全に開いた状態) まで段階的に変化させる ことによって、ウィンドウが開いたり閉じたりするアニメーションを行います。オープン度が 255 未満のとき、ウィンドウの内容は表示されません。初期値は 255 です。

### tone (RGSS3)



ウィンドウ背景の色調 ([Tone](gc_tone.md)) です。
