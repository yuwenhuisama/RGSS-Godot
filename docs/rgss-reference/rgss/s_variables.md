# 組み込み変数


組み込み変数はグローバル変数の一種ですが、その値はグローバルとは 限りません。

以下、変数の値のスコープで大別していくつかの組み込み変数を列挙します。

## ローカルスコープ



### $~



現在のスコープで最後に成功したマッチに関する情報 ([MatchData](sc_matchdata.md) オブジェクト) 。 これを設定すると $& や $1 ... $9 などの 値が変化します。

[Regexp.last_match](sc_regexp.md#L001314) と同じ。

### $&



現在のスコープで最後に成功した正規表現のパターンマッチでマッチした 文字列。最後のマッチが失敗していた場合には nil 。

[Regexp.last_match](sc_regexp.md#L001314)[0] と同じ。

### $1

### $2

### $3 ...



最後に成功したパターンマッチで n 番目の括弧にマッチした値が格 納されます。該当する括弧がなければ nil が入っています。

[Regexp.last_match](sc_regexp.md#L001314)[1], [Regexp.last_match](sc_regexp.md#L001314)[2], ... と同じ。

## グローバルスコープ



### $!



最近の例外に関する情報。[raise](s_functions.md#L000388) によって設定されま す。

### $DEBUG



デバッグモードフラグ。
