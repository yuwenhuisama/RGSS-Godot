# データベースの編集


## データベースとは


データベースとは、マップとマップイベント以外のゲームの構成要素をまとめたものです。プレイヤーが操作するキャラクターやアイテムや魔法などの要素や設定を、下記の14種類のデータとして用意できるようになっています。

［システム］と［用語］を除くデータは、ゲームに登場させる単位ごとにデータを用意します。たとえば“回復薬”や“カギ”などの道具類は、物品ごとに［アイテム］のデータを作り、それぞれ名前やグラフィック、使用時の効果などの特徴を設定します。

プロジェクトに取り込んだ素材ファイルも、用途ごとのデータに割り当てることで、ゲーム上に表示・再生できる仕組みです。

| アクター | プレイヤーが操作可能なキャラクター |
| --- | --- |
| 職業 | アクターに付加する特徴（能力値、成長条件など） |
| スキル | 戦闘時の行動や特殊技能（通常攻撃、特技や魔法など） |
| アイテム | プレイヤーが所持できる物品（回復薬や鍵など） |
| 武器 | アクターの攻撃用の装備品（剣や杖、弓など） |
| 防具 | アクターの防御用の装備品（鎧、盾、アクセサリーなど） |
| 敵キャラ | アクターの戦闘相手となるキャラクター |
| 敵グループ | 敵キャラの登場単位 |
| ステート | アクターなどの状態（毒、戦闘不能など） |
| アニメーション | スキルの使用時などに表示する視覚効果 |
| タイル | マップの作成に用いるタイルセットの構成 |
| コモンイベント | 汎用的に実行するイベント処理 |
| システム | 初期設定や音楽などゲーム全体の基本設定 |
| 用語 | ゲーム共通のコマンドや設定項目の名称 |



## データベースの編集方法



### ●編集画面の表示


データベースの内容は［データベース］のウィンドウで編集します。ウィンドウを表示するには、ツールバーの［データベース］のボタン（またはメニューの［ツール］→「データベース」）をクリックします。

### ●編集画面の構成


［データベース］ウィンドウの画面は、データの種類ごとのタブに分けられています。このタブをクリックして画面を切り替えることで、それぞれのデータを作成／編集できるようになります。

［システム］と［用語］以外のデータを編集する際は、まずウィンドウ左側のデータリストで編集するデータをクリックで選択します。そのうえで、ウィンドウ右側に並ぶ設定項目の内容を編集することでデータの特徴を定義します。

データリストで選択するデータは、［F4］［F5］キーで前後のデータに切り替えることができます。

### ●データ数の変更


［システム］［用語］以外のデータは最大999点まで用意できます。データ数を変更するには、データリストの下にある［最大数の変更］をクリックし、データ数（1～999）を指定します。最大数を減らした場合、それを超える番号のデータはすべて削除されます。

### ●コンテキストメニューによる操作


データリストの項目を右クリックするとメニューが表示され、そのデータを対象にコピーなどの操作が行なえます。各項目の操作の内容は以下のとおりです。

### コピー


データの設定内容をクリップボードに取り込みます。

### 貼り付け


クリップボード上の設定内容をデータに適用（上書き）します。操作前のデータの設定内容はすべて失われます。［マルチコピー］で複数のデータをクリップボードに取り込んでいる場合、右クリックしたもの以降のデータに設定内容が一括適用されます。

### クリア


データの設定内容をすべて消去します。

### マルチコピー


複数のデータの設定内容をまとめてクリップボードに取り込みます。右クリックした項目から取り込むデータの点数を指定します。


### ●データのIDについて


データリストの各項目の頭に表示される数字は、データのID（固有番号）を表わします。このIDはイベントコマンドなどで処理の対象とするデータを変数で指定するときなどに用います。

### ●メモ欄の活用


一部のデータ（アクター／職業／スキル／アイテム／武器／防具／敵キャラ／ステート／タイルセット）の設定項目には［メモ］の欄が用意されています。これはゲーム作成時のメモを自由に書き込むためのものです。プレイするゲームには一切影響を与えません。
