VX Ace では、多くの機能が能力値として設定されています。 ダメージ計算式はスキルやアイテムごとに設定されるため、決まった式は ありません。

## 通常能力値


通常能力値は、職業の［能力値成長曲線］で（敵キャラの場合は 数値で直接）設定された値を基本値とし、特徴リストの設定によって 乗算された値です。

| 通常能力値 | 略称 | 説明 |
| --- | --- | --- |
| 最大HP | MHP | HP の最大値です。 |
| 最大MP | MMP | MP の最大値です。 |
| 攻撃力 | ATK | 主に物理攻撃の与ダメージ量に影響します。 |
| 防御力 | DEF | 主に物理攻撃の被ダメージ量に影響します。 |
| 魔法力 | MAT | 主に魔法攻撃の与ダメージ量に影響します。 |
| 魔法防御 | MDF | 主に魔法攻撃の被ダメージ量に影響します。 |
| 敏捷性 | AGI | ターン内の行動順序を決定する値です。 |
| 運 | LUK | ステート付加、能力弱体の確率に影響します。 |



攻撃力、防御力、魔法力、魔法防御の 4 種類は、スキルやアイテムに 設定されたダメージ計算式で参照される以外に、特別な効果はありません。


運による、ステートおよび弱体有効度の変化率は次の通りです。ただし 0 を下限とします。
変化率 (%) = 100 + (使用者の運 - 対象者の運) ÷ 10

## 追加能力値


追加能力値は初期値を 0% とし、特徴リストの設定によって 加算された値です。

| 追加能力値 | 略称 | 説明 |
| --- | --- | --- |
| 命中率 | HIT | 物理攻撃が対象に命中する確率です。 |
| 回避率 | EVA | 物理攻撃を回避できる確率です。 |
| 会心率 | CRI | 会心の一撃が発生する確率です。 |
| 会心回避率 | CEV | 会心の一撃を防止する確率です。 |
| 魔法回避率 | MEV | 魔法攻撃を打ち消す確率です。 |
| 魔法反射率 | MRF | 魔法攻撃を跳ね返す確率です。 |
| 反撃率 | CNT | 物理攻撃に対して反撃を行う確率です。 |
| HP再生率 | HRG | 毎ターン終了時に回復する HP の割合です。 |
| MP再生率 | MRG | 毎ターン終了時に回復する MP の割合です。 |
| TP再生率 | TRG | 毎ターン終了時に増加する TP の割合です。 |



物理攻撃に関する命中判定と回避判定は独立して行われます。 たとえば攻撃側の命中率が 90%、対象側の回避率が 10% のとき、 回避率は 90% の中の 10% ですから、全体としての命中率は 81% 相当という計算になります。

反撃率は敵からの物理攻撃に対してのみ有効です。

## 特殊能力値


追加能力値は初期値を 100% とし、特徴リストの設定によって 乗算された値です。

| 特殊能力値 | 略称 | 説明 |
| --- | --- | --- |
| 狙われ率 | TGR | 敵キャラからの攻撃対象になる比率です。 |
| 防御効果率 | GRD | 防御コマンドのダメージ軽減効果の強さです。 |
| 回復効果率 | REC | HP、MP の回復効果を受ける割合です。 |
| 薬の知識 | PHA | アイテムによる HP、MP の回復を強化する割合です。 |
| MP消費率 | MCR | スキルの消費 MP を変動させる割合です。 |
| TPチャージ率 | TCR | スキル、アイテムの得 TP を変動させる割合です。 |
| 物理ダメージ率 | PDR | 物理ダメージを受ける割合です。 |
| 魔法ダメージ率 | MDR | 魔法ダメージを受ける割合です。 |
| 床ダメージ率 | FDR | マップ上の地形によるダメージを受ける割合です。 |
| 経験獲得率 | EXR | 獲得する経験値の量を変動する割合です。 |




防御効果率によるダメージ軽減割合は次の通りです。
防御時ダメージ = 通常ダメージ ÷ (2 × 防御効果率)

## 先制攻撃と不意打ち



### アクターの敏捷性平均値 >= 敵キャラの敏捷性平均値 のとき

先制攻撃率 (%) = 5
不意打ち率 (%) = 3

### アクターの敏捷性平均値 < 敵キャラの敏捷性平均値 のとき

先制攻撃率 (%) = 3
不意打ち率 (%) = 5

- 先制攻撃の場合は 1 ターン目に敵キャラ側が行動できなくなります。
- 不意打ちの場合は 1 ターン目にアクター側が行動できなくなります。
- イベント戦闘では、先制攻撃や不意打ちは発生しません。
- パーティ能力［先制攻撃率アップ］が有効の場合、確率は通常の 4 倍となります。
- パーティ能力［不意打ち無効］が有効の場合、不意打ちは発生しません。


## 逃走成功率
 逃走成功率 (%) = 150 - 100 × 敵キャラの敏捷性平均値 ÷ アクターの敏捷性平均値

- 逃走失敗 1 回ごとに 10% を加算します。
- 先制攻撃中は無条件で逃げられるものとします。
