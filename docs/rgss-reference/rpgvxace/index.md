 **RPGツクールVX Ace ヘルプ**


本ヘルプでは、『RPGツクールVX Ace』に搭載された機能についての説明を掲載しています。

このウインドウの左側が、ヘルプの目次となっています。ここから見たい項目を選んでください。調べたい項目が明らかなときは、左上にあるタブを［キーワード］や［検索］に切り替えることで探しやすくなります。 ![ヘルプタブ](img/0000_01.png)

また、本ソフトウェアでは、ほとんどの項目にポップアップヒントが用意されています。エディタ上で項目にマウスカーソルを合わせて少し待つと、簡単な説明が表示されますので、こちらもご活用ください。 ![ポップアップヒント](img/0000_02.png)
