## ［命中率］の設定方法


武器の命中率の設定方法です。

［武器］特徴 － 能力値 － 追加能力値 － 命中率

![](img/010/010_002d_01.png)

- VX Ace では、基本的にアクターや職業に命中率が設定されていますので、その**命中率を補正する数値を設定する**ようにしてください。
- 命中率を設定しない場合は、アクターや職業に設定された命中率がそのまま適用されます。
