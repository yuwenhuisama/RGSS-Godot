# 『RPGツクールVX』ユーザー向けの情報


『RPGツクールVX Ace』（以下VX Ace）は、前作『RPGツクールVX』（以下VX）の正統進化バージョンですので、多数の新機能が追加されたとともに、VX で出来たことは VX Ace でもほとんど出来るようになっています。しかしながら、データベースの設定に関しては VX から設定方法が変わった機能も多数ありますので、ここではそういった機能と、VX ユーザーが戸惑いやすいポイントについて解説します。
